# Expense Tracker (Python)

A simple Expense Tracker built using Python.

## Features

- Add Expense
- View Expenses
- Search Expense
- Edit Expense
- Delete Expense
- Calculate Total Expense
- File Handling using expenses.txt
- Exception Handling (try-except)

## Technologies Used

- Python
- File Handling
- Functions
- Lists
- Loops

## How to Run

1. Download the project.
2. Open expense_tracker.py in Pydroid 3 or VS Code.
3. Run the program.
4. Follow the menu options.
